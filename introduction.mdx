# Welcome to Fusion GMBP

Fusion GMBP is a Gaussian Mixture Belief Propagation implementation for probabilistic inference on continuous variables with non-linear, multi-modal relationships.

## What is GMBP?

Unlike standard Gaussian Belief Propagation (GaBP), which assumes linear relationships and unimodal distributions, GMBP uses Gaussian Mixtures to preserve ambiguity throughout the inference process.

For example, if X is observed, Y could be 5 OR 10 - GMBP maintains both possibilities as separate components in the mixture, enabling true multi-modal reasoning.

## Core concepts

### Factor graphs

Fusion GMBP models probabilistic systems as factor graphs with:
- **Variables**: Continuous random variables you want to infer
- **Factors**: Learned joint probability distributions (Gaussian Mixtures) over variable subsets
- **Evidence**: Observed values that constrain the inference

### Belief propagation

The library uses message-passing belief propagation to compute posterior beliefs:
1. Initialize messages between variables and factors
2. Iteratively update messages using GMM operations (product, conditioning, marginalization)
3. Compute posterior beliefs by combining all incoming messages
4. Monitor convergence with multiple diagnostics

### Gaussian mixtures

Each message and belief is represented as a Gaussian Mixture Model:

```python
GMMContainer(
    weights=[0.6, 0.4],           # Component weights
    means=[[5.0], [10.0]],        # Component means
    covs=[[[0.5]], [[0.8]]],      # Component covariances
    var_names=["Y"]               # Variable names
)
```

This represents a bimodal distribution where Y could be around 5 (60% weight) or around 10 (40% weight).

## Architecture

Fusion GMBP is designed with modularity in mind:

- **CLI interface**: `gmbp` command with `fit`, `run`, `analyze` subcommands
- **Python API**: Direct access to `BPEngine`, `GMMContainer`, and GMM operations
- **Web visualizer**: Interactive step-by-step inference exploration
- **Analysis tools**: Complexity sweeps, validation splits, and model selection
